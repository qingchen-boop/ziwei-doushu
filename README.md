# Zi Wei Dou Shu - Zi Wei Dou Shu Fortune Charting System

High-precision Zi Wei Dou Shu charting system with support for true solar time calculation, North/South sect algorithms, and Four Transformations linkage.

## Tech Stack

- **Frontend**: Next.js 14 (React 18)
- **Type System**: TypeScript
- **Styling**: Tailwind CSS
- **State Management**: Zustand
- **Calendar Calculation**: lunar-javascript
- **Deployment**: GitHub Pages

## Features

- True solar time calculation (longitude/latitude compensation)
- Early/Late Zi Shi logic toggle
- North/South sect charting algorithms
- Four Transformations (Si Hua) calculation
- Liu Nian/Da Yun Four Transformations linkage
- Professional Dark / Modern Light themes
- PC Three-Column Layout / Mobile Nine-Grid

## Development

```bash
# Install dependencies
npm install

# Development mode
npm run dev

# Type check
npm run typecheck

# Build
npm run build
```

## Deploy

Push to `main` branch to trigger GitHub Pages deployment.
